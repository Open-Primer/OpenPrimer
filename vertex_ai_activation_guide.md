# Activation de l'API Vertex AI (Compte Professionnel)

Pour garantir que l'API est activée sur le compte disposant des 300$ de crédits, veuillez suivre ces étapes précises :

### 1. Accès à la Console avec Sélecteur de Compte
Utilisez ce lien qui force le sélecteur de projet pour l'ID spécifique :
[https://console.cloud.google.com/apis/library/aiplatform.googleapis.com?project=project-31f1d0d7-25e7-46c9-ad5](https://console.cloud.google.com/apis/library/aiplatform.googleapis.com?project=project-31f1d0d7-25e7-46c9-ad5)

### 2. Vérification du Compte (Haut à droite)
Une fois sur la page :
- **Vérifiez l'icône de profil** en haut à droite. 
- Si ce n'est pas le compte avec les crédits, cliquez sur l'icône et sélectionnez le bon compte.
- **IMPORTANT** : Si vous changez de compte, assurez-vous que le projet `project-31f1d0d7-25e7-46c9-ad5` est toujours sélectionné dans la barre bleue en haut.

### 3. Bouton "ACTIVER"
- Cliquez sur le bouton bleu **ACTIVER**.
- Si le bouton affiche "GÉRER", l'API est déjà active.
