# 4. Évaluations, Quiz & Feedback Interactifs

Bienvenue dans la quatrième et dernière leçon de la Sandbox de Test d'OpenPrimer ! Cette page rassemble **l'ensemble des activités d'évaluation, de quiz formatif, d'entraînement à l'oral et à l'écrit par IA, d'analyse socratique et de jeux d'association**.

## 🏋️ Exercices d'Entraînement Pratiques

### 🔓 Exercice Corrigé (SolvedExercise)
Un bloc structuré présentant un problème complexe dont l'étudiant peut révéler la solution rédigée de manière pas-à-pas en cliquant sur un bouton d'expansion.

<SolvedExercise title="Calcul de la complexité d'une boucle log">
  Soit le code suivant où $N$ est un entier positif :
  ```cpp
  int i = 1;
  while (i < N) {
      i = i * 2;
  }
  ```
  Déterminez la complexité temporelle de cet algorithme en notation Grand O.
  
  <Solution>
    À chaque itération de la boucle, la variable `i` est doublée :
    - Itération 0 : `i = 1`
    - Itération 1 : `i = 2`
    - Itération 2 : `i = 4`
    - Itération $k$ : `i = 2^k`
    
    La boucle s'arrête dès que `i >= N`, c'est-à-dire quand $2^k \ge N$.
    En prenant le logarithme de chaque côté de l'inéquation, nous obtenons :
    $$k \ge \log_2(N)$$
    
    Le nombre d'itérations de la boucle est donc directement proportionnel au logarithme de $N$. 
    Par conséquent, la complexité temporelle est **logarithmique**, soit $\mathcal{O}(\log N)$.
  </Solution>
</SolvedExercise>

### 🔒 Exercice d'Application Directe (UnsolvedExercise)
Un exercice interactif demandant à l'apprenant de saisir sa réponse numérique ou textuelle. Une validation automatique est effectuée à la validation avec des indices facultatifs et une tolérance numérique paramétrable.

<UnsolvedExercise 
  title="Capacité de Canal (Formule de Shannon)" 
  question="Un canal de communication sans fil possède une largeur de bande de 3000 Hz et un rapport signal/bruit (S/N) de 31. D'après le théorème de Shannon-Hartley, quelle est la capacité maximale théorique de ce canal en bits par seconde (bps) ?" 
  correctAnswer={15000} 
  unit="bps" 
  placeholder="Saisissez la valeur numérique..." 
  hint="Utilisez la formule : C = B * log2(1 + S/N). Ici, S/N vaut 31, donc (1 + S/N) vaut 32. log2(32) est une valeur entière simple." 
  solution="La formule de Shannon est C = B * log2(1 + S/N). En remplaçant par les valeurs données : C = 3000 * log2(1 + 31) = 3000 * log2(32). Puisque 2^5 = 32, log2(32) = 5. On a donc C = 3000 * 5 = 15000 bps." 
/>
