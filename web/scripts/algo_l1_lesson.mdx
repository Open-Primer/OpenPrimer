## Introduction à la Pensée Algorithmique

Un algorithme est une suite d'instructions précises, ordonnées et finies permettant de résoudre un problème ou d'accomplir une tâche. C'est en quelque sorte la recette de cuisine de l'informaticien. Dans cette première leçon de niveau L1, nous allons explorer les briques de base de tout algorithme : les variables, les structures conditionnelles et les boucles de répétition.

### 1. Les Variables et les Types de Données

En informatique, une **variable** est une boîte nommée dans la mémoire de l'ordinateur où l'on peut stocker une valeur pour l'utiliser plus tard. Chaque variable possède un **nom** (un identifiant), un **type** (qui détermine la nature de ce qu'elle contient) et une **valeur**.

Les types de données les plus courants sont :
*   **Nombre** (Integer/Float) : représentant des entiers ou des réels (ex: `42`, `3.14`).
*   **Chaîne de caractères** (String) : représentant du texte (ex: `"Bonjour"`).
*   **Booléen** (Boolean) : représentant une valeur de vérité, soit `true` (vrai) soit `false` (faux).

### 2. Les Structures Conditionnelles (Si... Alors... Sinon)

Un programme informatique ne s'exécute pas toujours de manière strictement linéaire. Souvent, nous devons prendre des décisions basées sur des conditions. C'est le rôle de l'instruction conditionnelle.

En pseudo-code :
```
Si condition Alors
    instructions exécutées si la condition est vraie
Sinon
    instructions exécutées si la condition est fausse
FinSi
```

En JavaScript, cela s'écrit :
```javascript
if (age >= 18) {
    console.log("Vous êtes majeur.");
} else {
    console.log("Vous êtes mineur.");
}
```

---

### 3. Les Boucles de Répétition

Les boucles permettent d'exécuter un bloc d'instructions plusieurs fois d'affilée sans avoir à le réécrire.

*   La boucle **Tant Que (while)** : répète les instructions tant qu'une condition reste vraie.
*   La boucle **Pour (for)** : répète les instructions un nombre de fois prédéterminé, généralement à l'aide d'un compteur.

#### Exemple de boucle pour afficher les nombres de 1 à 5 :
```javascript
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
```
