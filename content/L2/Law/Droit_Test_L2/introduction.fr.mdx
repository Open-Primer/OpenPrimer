# Introduction au Droit Test L2 : Droit Pénal

Bienvenue dans le cours **Droit Test L2**. Ce cours de test avancé explore les éléments constitutifs des infractions pénales et de la responsabilité.

<SelfEval title="Introduction au Droit Test L2" type="pre" />

## 1. Principes de la Responsabilité Pénale
Une infraction se compose de deux éléments fondamentaux : l'acte matériel (*actus reus*) et l'intention morale (*mens rea*).

<MetaNote title="Concept Clé">
  Les deux éléments doivent concourir pour établir la responsabilité pénale dans les systèmes pénaux modernes.
</MetaNote>
