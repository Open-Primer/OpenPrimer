# Contributing to OpenPrimer

Thank you for your interest in democratizing universal knowledge! OpenPrimer is a community project where AI and Humans collaborate.

## How to Contribute?

### 1. Suggest a new curriculum
If you think a subject is missing from the catalog (e.g., L1 Cognitive Psychology), open an **Issue** with the `course-request` label. The AI will start working on a syllabus as soon as the request is validated.

### 2. Report or correct an error
The easiest way is to use the **"Report"** button directly on the site. If you prefer the classical method:
- Open a **Pull Request** on the relevant `.mdx` file in the `content/` folder.
- The "Correction" AI will analyze your proposal before final human validation.

### 3. Improve the interface or agents
All source code (Next.js, Python) is open. Check `docs/ARCHITECTURE.md` to understand how the engine works.

## 🤝 Code of Conduct
Academic rigor is our priority. All contributions must be sourced and respect university standards.
